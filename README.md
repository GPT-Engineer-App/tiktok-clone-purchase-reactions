## Setup

```sh
git clone https://github.com/GPT-Engineer-App/tiktok-clone-purchase-reactions.git
cd tiktok-clone-purchase-reactions
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
